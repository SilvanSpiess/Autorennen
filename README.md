# Autorennen
Topview Autorennen für 2 Spieler. Ehemaliges Informatik Projekt aus dem Gymnasium (Version 1.0), welche verbessert wurde in 3.0.

---

## Build
Both versions of the game can be built using the corresponding commandline files. Builds exist for Windows and MacOS.

Important, the build must be completed in an environment with the following libraries installed:
- pygame
- numpy
- math
- time
- datetime
- random

Windows:
```bash
.\build_X_0_Win.bat
```

MacOS
```sh
sh \<drag buidl file here\>
```
